=======
障害URL
=======

概要
====

ここでは、障害URLについて説明します。クロール時に取得できなかった URL
が記録され、障害 URL として確認することができます。

設定方法
========

表示方法
--------

管理者アカウントでログイン後、メニューの障害 URL をクリックします。

|image0|

障害 URL の確認リンクをクリックすると、詳細が表示されます。

|image1|

一覧
----

クロールできなかった URL と日時が一覧できます。
